Installation
-------

This package can be installed via [Composer]:

``` bash
$ composer require jedkirby/tweet-entity-linker
```

It requires **PHP >= 5.6.4**.

Testing
-------

Unit tests can be run inside the package:

``` bash
$ ./vendor/bin/phpunit
```
